# 重要事务记录 & 闹钟提醒

一个简洁美观的任务管理网站，支持闹钟提醒功能。

## ✨ 功能特点

- 📋 **任务管理** - 添加、编辑、删除任务
- 🎯 **优先级设置** - 高、中、低三级优先级
- ⏰ **闹钟提醒** - 浏览器通知提醒功能
- 📅 **日历导出** - 支持导出到 Google/Outlook 日历
- 📱 **响应式设计** - 支持移动端访问
- 💾 **本地存储** - 数据保存在浏览器本地

## 🎯 使用方法

1. 点击"添加任务"按钮创建新任务
2. 设置任务标题、描述、优先级和截止时间
3. 允许浏览器通知权限以接收提醒
4. 右上角按钮可以测试通知功能

## 🔔 关于提醒功能

- 首次使用需要允许浏览器通知权限
- 页面需要保持打开状态才能接收提醒
- 点击右上角"⏰ 测试提醒"可以测试功能

## 📝 技术栈

- HTML5 + CSS3 + JavaScript
- Web Notification API
- LocalStorage API
- Web Audio API
